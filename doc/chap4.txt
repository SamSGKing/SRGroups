  [1X4.2-1 SRGroupFile[101X
  
  [33X[1;0Y[29X[2XSRGroupFile[102X( [3Xk[103X ) [32X function[133X
  
  [33X[0;0YThe  arguments  of  this  function are a degree, [3Xk[103X, or [3X0[103X. If the argument is
  non-zero,  this  function  creates  the file containing all self-replicating
  groups  of  the  regular rooted k-tree at the lowest level not stored in the
  [5XSRGroups[105X  library.  If  the  argument  is  [3X0[103X, this function creates the file
  containing  all  self-replicating  groups  of the regular rooted tree at the
  level  1  for the lowest degree not stored in the [5XSRGroups[105X library. The file
  naming  convention is "sr_k_n.grp", and they are stored in the "data" folder
  of  the  [5XSRGroups[105X  package. Level 1 groups are calculated using the [5Xtransgrp[105X
  library.  If the argument is non-zero and there is a gap between files (i.e.
  if  "sr_k_n.grp"  and  "sr_k_n+2.grp"  exists,  but  "sr_k_n+1.grp" does not

  [33X[1;0Y[29X[2XExtendSRGroup[102X( [3Xarg[103X ) [32X function[133X
  
  [33X[0;0YThe  arguments  of  this  function are: arg[1]: degree of tree (int > 1), [3Xk[103X,
