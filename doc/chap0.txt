  [1XAbstract[101X
  [33X[0;0Y[5XSRGroups[105X  is  a  package for searching up self-replicating groups of regular
  rooted  trees  and  performing  computations  on  these groups. This package
  allows  the  user to generate more self-replicating groups at greater depths
  with its in-built functions, and is an extension of the [5Xtransgrp[105X package.[133X
